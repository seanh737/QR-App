# QR-App

## Project Install

```sh
git clone https://github.com/seanh737/QR-App.git

cd ./QR-App

yarn or npm i
```
Run on IOS :
```sh
react-native run-ios
```
Run on Android ( Remember to add `local.properties` in file in android's folder ) :
```sh
react-native run-android
```

## Git 指令
當電腦版本較舊
```sh
git pull --rebase
```
上傳
```sh
git push -u origin master
```
